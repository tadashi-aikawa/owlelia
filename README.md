## For developers

### Setup

```bash
git config core.hooksPath hooks
```

### Requirements

- Node.js >= 22
- TypeScript >= 5.6

### Develop

```bash
pnpm install
pnpm build
pnpm test
```

### Release

Run [Release Action](https://github.com/tadashi-aikawa/owlelia/actions/workflows/release.yaml) manually.
